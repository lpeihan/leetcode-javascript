## Environment

```bash
# 1
npm init -f

# 2. 安装 eslint
npm install eslint
npx eslint --init

# 3. 安装 jest 环境，支持 es6
npm install jest
jest --init
npm install babel-jest @babel/core @babel/preset-env @babel/preset-typescript -D
```
